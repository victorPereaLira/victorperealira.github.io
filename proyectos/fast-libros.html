Fast Libros

Fast Libros es una aplicación web desarrollada en Python que permite gestionar tu biblioteca de forma ágil y desde cualquier navegador. Está pensada para facilitar la organización de colecciones de libros, con una interfaz simple y funcional.

✨ Características principales

  • 📂 Registro y gestión de libros con título, autor y categoría.
  • 🔎 Búsqueda rápida de libros.
  • 🗄️ Base de datos integrada para almacenamiento seguro.
  • 🌍 Acceso desde cualquier dispositivo gracias a la interfaz web.

⚙️ Tecnologías utilizadas

Python Flask SQL HTML & CSS
