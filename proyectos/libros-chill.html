Libros & Chill

Libros & Chill es una aplicación Android desarrollada para organizar tu biblioteca personal de forma sencilla y visual. Con ella nos permite gestionar nuestros libros, añadir nuestra opinion y mantener un registro de nuestras lecturas.

✨ Características principales

  • 📚 Gestión de libros favorios, leidos y pendientes.
  • 📝 Añadir comentarios sobre el libro pudiendo leer los de mas usuarios.
  • 🔎 Búsqueda rápida de libros por título o autor.
  • ☁️ Integración con Firebase para sincronizar datos.
  • ☁️ Impementacion de la API de Google Books para facilitar la busqueda y añadir libros a nuestra biblioteca.
  • 🎨 Interfaz intuitiva y atractiva basada en Material Design.

⚙️ Tecnologías utilizadas

Kotlin Firebase Material Design API de Libros
